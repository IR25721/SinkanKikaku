# 分布体験会 (SinkanKikaku)

「分布体験会」は、大学などの新歓企画（オープンキャンパス等）にて、様々な確率分布の性質を直感的に体験し、データサイエンスや統計学の面白さに触れてもらうためのインタラクティブなWebアプリケーションです。

## 概要 (Overview)

参加者が簡単なゲーム（例えば体内時計で10秒を計る、コイントスをするなど）をプレイすることで、その結果がどのような確率分布に従うのかをリアルタイムでシミュレーション・可視化します。

**主な機能:**
1. **10種類の確率分布シミュレーション**:
   - **基本的な分布**: 正規分布、指数分布、ポアソン分布、一様分布
   - **応用的な分布**: 対数正規分布、幾何分布、ベータ分布
   - **発展的な分布**: コーシー分布、ワイブル分布、ガンマ分布
2. **リアルタイムな可視化**: 
   - Chart.jsを用いたヒストグラムの即時描画と理論PDFのオーバーレイ
   - ベイズ更新プロセスの可視化（ベータ分布など）
3. **情報幾何学 (Information Geometry) シミュレータ**: 
   - 確率分布の集合に計量を導入した「統計多様体」を体感するためのシミュレータ。
   - 2つの正規分布間を結ぶ最短距離（測地線）をFisher情報量に基づく双曲幾何学上で正確に計算し、分布空間が「歪んでいる」様子をアニメーションで確認できます。

## 技術スタック (Tech Stack)

- **フロントエンド**: HTML, Vanilla JavaScript, Vanilla CSS (カスタムデザインシステム)
- **ビルドツール**: Vite
- **可視化ライブラリ**: Chart.js
- **統計・数学計算**: jStat (各種PDFの計算用), 自作の数値計算モジュール
- **数式レンダリング**: KaTeX (CDN読み込み)

## 開発と実行 (Setup & Build)

ローカルで開発・実行するための手順です。実行には [Node.js](https://nodejs.org/) が必要です。

### 1. 依存パッケージのインストール
```bash
npm install
```

### 2. 開発サーバーの起動
ローカルで変更を監視しながらアプリを動かくす場合（デフォルトで `http://localhost:5173` 等で起動します）。
```bash
npm run dev
```
（外部デバイスからアクセスして体験してもらう場合は `npm run dev -- --host 0.0.0.0` のようにホスト指定を行ってください）

### 3. テストの実行
Vitest フレームワークを用いてフロントエンドロジックの単体テストを実行します。
```bash
npm run test
```

### 4. 本番用ビルド
プロダクション用にビルド（最適化・バンドル）する場合は以下のコマンドを実行します。
```bash
npm run build
```
ビルド完了後、`dist` ディレクトリに生成された静的ファイルを任意のWebサーバー（GitHub Pages, Vercel, Netlify など）にデプロイするだけで公開可能です。

## 機能ハイライト
- **アカデミックなUI**: デザインや文言は学術的なトーン（句読点 `. ,`の統一など）で統一されており、数学の数式（$N(\mu, \sigma^2)$ 等）もインラインLaTeXでレンダリングされます。
- **データのエクスポート**: 体験後、リセットボタンを押すことで各ゲームの生データをCSVとして安全にダウンロード・保存できます。
